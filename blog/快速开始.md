# 博客快速开始指南

## 环境准备

确保已安装:
- Node.js 14+ 
- Git
- npm

## 安装步骤

```bash
# 进入博客目录
cd d:\大四下\博客修复与重整\blog

# 安装依赖
npm install

# 安装渲染器
npm install hexo-renderer-pug hexo-renderer-stylus --save

# 安装字数统计
npm install hexo-wordcount --save

# 克隆主题(如果还没有)
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly
```

## 本地预览

```bash
# 清理缓存
npx hexo clean

# 生成静态文件
npx hexo generate

# 启动本地服务器
npx hexo server
```

访问 http://localhost:4000 查看效果

## 添加新文章

```bash
# 创建新文章
npx hexo new "文章标题"

# 编辑文章
编辑 source/_posts/文章标题.md

# 重新生成
npx hexo generate
```

## 部署到 GitHub Pages

```bash
# 生成并部署
npx hexo clean && npx hexo generate && npx hexo deploy
```

## 配置说明

主要配置文件:
- `_config.yml` - Hexo 主配置
- `_config.butterfly.yml` - 主题配置

个性化设置需要修改这两个文件。

## 常用命令速查

| 命令 | 说明 |
|------|------|
| `npx hexo new "title"` | 新建文章 |
| `npx hexo new page "about"` | 新建页面 |
| `npx hexo clean` | 清理缓存 |
| `npx hexo generate` | 生成静态文件 |
| `npx hexo server` | 启动本地服务器 |
| `npx hexo deploy` | 部署 |

## 注意事项

1. 首次使用需要在 `source/img/` 目录放置头像和图片
2. 修改配置后需要重新生成静态文件
3. 部署前确保 GitHub 仓库和 SSH 密钥配置正确
4. 详细配置请参考《博客修改手册.md》
