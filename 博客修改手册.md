# 博客重整修改手册

## 项目概述

本项目是将高中时期使用的 Hexo 博客(基于 yilia 主题)升级为最新的 Hexo 7.x 版本,并采用 Butterfly 现代化主题进行全面重构。

## 一、升级内容

### 1.1 Hexo 版本升级
- **原版本**: Hexo 5.4.0
- **新版本**: Hexo 7.3.0
- **升级方式**: 修改 `package.json` 依赖版本并重新安装

### 1.2 主题更换
- **原主题**: yilia (较旧的版本)
- **新主题**: Butterfly 5.5.4 (2025年最流行的 Hexo 主题之一)
- **主题特点**:
  - 现代化卡片式设计
  - 响应式布局,完美适配移动端
  - 丰富的配置选项
  - 支持深色模式
  - 内置搜索功能
  - 侧边栏小部件丰富

### 1.3 依赖更新
主要更新了以下核心插件:
- `hexo` 5.0.0 → 7.3.0
- `hexo-deployer-git` 3.0.0 → 4.0.0
- `hexo-generator-*` 系列全部升级到最新版本
- `hexo-renderer-marked` 4.0.0 → 6.0.0
- 新增 `hexo-wordcount` 字数统计插件
- 新增 `hexo-theme-butterfly` 5.3.0

## 三、安装步骤

### 3.1 环境要求
- Node.js 14.x 或更高版本
- Git
- npm

### 3.2 安装命令

```bash
# 1. 进入博客目录
cd d:\大四下\博客修复与重整\blog

# 2. 安装依赖
npm install

# 3. 安装 Butterfly 主题所需的渲染器
npm install hexo-renderer-pug hexo-renderer-stylus --save

# 4. 安装字数统计插件
npm install hexo-wordcount --save

# 5. 克隆 Butterfly 主题
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

# 6. 清理缓存
npx hexo clean

# 7. 生成静态文件
npx hexo generate

# 8. 启动本地服务器预览
npx hexo server
```

### 5.2 修改主题颜色

编辑 `_config.butterfly.yml` 中的 `theme_color` 配置:

```yaml

  enable: true
  main: "#49B1F5"  # 主题主色调
  paginator: "#00C4B6"  # 分页器颜色
  button_hover: "#FF7242"  # 按钮悬停色

  link_color: "#99a9bf"  # 链接颜色
  hr_color: "#A4D8FA"  # 分割线颜色
```

### 5.3 配置评论系统

在 `_config.butterfly.yml` 中配置评论插件(以 Valine 为例):

```yaml
comments:
  use: Valine
  valine:
    appId: your_app_id
    appKey: your_app_key
    avatar: monsterid
    serverURLs: ''
    bg: #valine_background
    enable: true
    visitor: true
    highlight: true
    pageSize: 10
    placeholder: 请留下你的评论...
    requiredFields:
    - nick
    - mail
```

## 六、部署到 GitHub Pages

### 6.1 配置 GitHub 仓库

1. 确保已有 GitHub 仓库 `60777flowers.github.io`
2. 确保 `_config.yml` 中的 deploy 配置正确

### 6.2 生成 SSH Key

```bash
# 生成 SSH 密钥
ssh-keygen -t ed25519 -C "your_email@example.com"

# 查看公钥并添加到 GitHub
cat ~/.ssh/id_ed25519.pub
```

### 6.3 部署命令

```bash
# 部署到 GitHub Pages
npx hexo deploy

# 或使用组合命令
npx hexo clean && npx hexo generate && npx hexo deploy
```

### 6.4 自动部署 (可选)

创建 GitHub Actions 工作流,在 `.github/workflows/deploy.yml`:

```yaml
name: Hexo Deploy

on:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v2
    - name: Use Node.js
      uses: actions/setup-node@v2
      with:
        node-version: '18'
    - name: Install dependencies
      run: npm install
    - name: Generate
      run: npx hexo generate
    - name: Deploy
      uses: peaceiris/actions-gh-pages@v3
      with:
        github_token: ${{ secrets.GITHUB_TOKEN }}
        publish_dir: ./public
```

## 七、常见问题解决

### 7.1 生成报错: "wordcount is not a function"

**解决方案**:
```bash
npm install hexo-wordcount --save
```

### 7.2 主题样式显示异常

**解决方案**:
```bash
npx hexo clean
npx hexo generate
```

### 7.3 部署失败

**检查项**:
1. Git 配置是否正确
2. SSH 密钥是否已添加到 GitHub
3. 仓库权限是否正确
4. 分支名称是否匹配

### 7.4 图片无法显示

**检查项**:
1. 图片路径是否正确(使用 `/img/xxx.png` 格式)
2. 图片文件是否存在于 `source/img/` 目录
3. 图片文件名大小写是否正确

## 八、进阶优化

### 8.1 开启图片懒加载

在 `_config.butterfly.yml` 中:

```yaml
lazyload:
  enable: true
  field: site # site/post
  placeholder:
  blur: true
```

### 8.2 配置 CDN 加速

在 `_config.yml` 中配置 CDN:

```yaml
# CDN 配置
CDN:
  css:
    - /css/index.css
  js:
    - /js/main.js
```

### 8.3 启用压缩

安装压缩插件:
```bash
npm install hexo-neat --save
```

在 `_config.yml` 中添加:

```yaml
neat_enable: true
neat_html:
  enable: true
  exclude:
neat_css:
  enable: true
  exclude:
    - '*.min.css'
neat_js:
  enable: true
  mangle: true
  output:
  compress:
  exclude:
    - '*.min.js'
```

### 8.4 添加 SEO 优化

安装 SEO 插件:
```bash
npm install hexo-generator-sitemap --save
npm install hexo-generator-baidu-sitemap --save
```

在 `_config.yml` 中添加:

```yaml
sitemap:
  path: sitemap.xml
baidusitemap:
  path: baidusitemap.xml
```
