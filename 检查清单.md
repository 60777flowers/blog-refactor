# 博客部署检查清单

## ✅ 需要检查的项目

### 1. 文件是否正确上传?

检查你的仓库 https://github.com/60777flowers/blog-refactor

应该包含:
- ✅ index.html (首页)
- ✅ 2021/ 和 2022/ 文件夹(文章)
- ✅ archives/ 文件夹(归档)
- ✅ tags/ 文件夹(标签)
- ✅ css/ 文件夹(样式)
- ✅ js/ 文件夹(脚本)
- ✅ img/ 文件夹(图片)

**如果没有这些文件,需要重新上传 `blog/public` 文件夹的内容**

---

### 2. GitHub Pages 是否已启用?

访问: https://github.com/60777flowers/blog-refactor/settings/pages

检查:
- Source 是否选择了 "Deploy from a branch"
- Branch 是否选择了 "main" 和 "/(root)"
- 是否点击了 "Save" 按钮
- 页面是否显示 "Your site is live at: https://60777flowers.github.io/blog-refactor/"

---

### 3. 能否访问博客?

尝试访问:
**https://60777flowers.github.io/blog-refactor/**

如果显示:
- ✅ 正常页面 → 成功!
- 🔄 "Your site is being deployed" → 再等 2-5 分钟
- ❌ "404 Not Found" → 检查步骤 1 和 2

---

## 🔍 常见问题

### 问题 1: 显示 404

**原因**: 
- GitHub Pages 未启用
- 文件未正确上传

**解决**:
1. 检查 Pages 设置是否正确
2. 确认上传了所有文件

### 问题 2: 样式错乱

**原因**: 
- 缺少 CSS 文件
- 路径不正确

**解决**:
1. 确认上传了 `css/` 文件夹
2. 检查浏览器控制台错误

### 问题 3: 图片不显示

**原因**: 
- 缺少 `img/` 文件夹
- 图片路径错误

**解决**:
1. 确认上传了 `img/` 文件夹
2. 检查 HTML 中的图片路径
